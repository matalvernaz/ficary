# Bundled soundscape loops

Drop short, seamless, CC0-licensed ambient loops here (`.ogg` preferred) to
ship them with Ficary. Reference them from a soundscape as `bundled:<name>`
(without extension) — e.g. a file `rain.ogg` here is `bundled:rain`.

None ship yet. Until then, add your own sound files through the soundscape
editor (Reader → Soundscape editor → Add sound); they're copied into the
`sounds/` folder under the Ficary data directory.
